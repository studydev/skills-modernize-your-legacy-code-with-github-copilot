# Copilot으로 레거시 코드 현대화

## 환영합니다

- **대상**: 레거시 코드베이스를 업데이트하고 개선하고자 하는 개발자
- **학습 내용**: GitHub Copilot을 활용하여 기존 코드를 리팩토링하고 현대화하는 방법
- **실습 내용**: 현대적인 코딩 관행을 보여주는 개선된 코드 스니펫 세트를 만듭니다.
- **사전 요구 사항**:
  - Skills 실습: [GitHub Copilot 시작하기](https://github.com/skills-kr/getting-started-with-github-copilot)
  - [VS Code](https://code.visualstudio.com/) 사용 경험
  - 기본 코딩 원리 이해

- **소요 시간**: 이 실습은 30분 이내에 완료할 수 있습니다.

이 실습에서는 다음을 수행합니다:

1. 레거시 코드의 현재 상태를 설명합니다.
1. 코드의 데이터 흐름 다이어그램을 만듭니다.
1. 개선할 수 있는 레거시 코드 영역을 식별합니다.
1. GitHub Copilot을 사용하여 현대적인 코드 스니펫을 생성합니다.
1. 이전 코드를 새 스니펫으로 교체하고 변경 사항을 테스트합니다.
